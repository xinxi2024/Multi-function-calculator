# 多功能计算器

这是一个多功能计算器，支持标准计算、程序员计算和日期计算。用户可以在不同的模式之间切换，进行各种计算。

## 功能

- **标准计算器**：支持基本的算术运算（加、减、乘、除、取余、幂）。
- **程序员计算器**：支持二进制、八进制、十进制和十六进制的运算，包括位运算（与、或、异或、非、左移、右移）。
- **日期计算器**：计算两个日期之间的天数，或在指定日期上添加或减去天数。

1. 打开 `index.html` 文件。
